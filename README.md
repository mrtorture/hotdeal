## 핫딜 공유 서비스
* Java, Spring 기반 CRUD 서비스 입니다.
* 사진을 포함한 게시글 CRUD, 페이징, 정렬, 필터링, 검색
* 가입, 로그인, 댓글/대댓글 CRUD, 추천, 좋아요 기능을 포함합니다.
