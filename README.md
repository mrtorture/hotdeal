## 핫딜 공유 서비스
* Java, Spring 기반 CRUD 서비스 입니다.
* 사진을 포함한 게시글 CRUD, 페이징, 정렬, 필터링, 검색 기능
* 가입, 로그인, 입력값 검증, 댓글/대댓글 CRUD, 좋아요, 찜하기 기능
